Data Security

How we protect your information

Since PDF Suite Pro operates completely offline and does not collect any user data, your information is inherently protected. However, we implement additional security measures to ensure your documents remain safe:

Security Measures

  • Local Processing Only: All PDF operations are performed entirely on your device using the device's native processing capabilities.
  • No Network Transmission: The app does not have internet permission and cannot transmit data outside your device.
  • Encrypted Storage: Password-protected PDFs use AES-256 encryption, the same standard used by banks and governments.
  • Scoped Storage: On Android 10+, the app uses scoped storage to limit access to only necessary files.
  • Temporary File Cleanup: Temporary processing files are automatically deleted after operations complete.

Privacy by Design: PDF Suite Pro was built from the ground up with privacy as a core principle. We believe the best way to protect your data is to never collect it in the first place.

Your Responsibilities

While we ensure the app doesn't compromise your data, we recommend:

  • Keeping your device secured with a PIN, password, or biometric lock
  • Using strong passwords when password-protecting sensitive PDFs
  • Regularly backing up important documents
  • Being cautious when sharing PDF files with others

Third-Party Services

External services and libraries used

PDF Suite Pro does not integrate with any third-party services that collect user data. The app operates independently without requiring internet connectivity.

Open-Source Libraries

The app uses open-source libraries for PDF processing, all of which operate locally:

  • PDFium: Open-source PDF rendering engine (runs locally)
  • iText: PDF manipulation library (runs locally)
  • AndroidX Libraries: Standard Android UI components
  • Glide: Image loading and caching (local only)

None of these libraries transmit data to external servers. All processing happens on your device.

No Advertising or Analytics

PDF Suite Pro does not contain:

  • Advertising SDKs (no ads)
  • Analytics or tracking services
  • Social media integration
  • Crash reporting services that upload data
